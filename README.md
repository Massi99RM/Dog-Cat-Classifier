# Cat vs Dog Image Classifier

A simple deep learning project that classifies images as either cats or dogs using PyTorch.
The images used for training and validation are from a 2013 "Dogs vs Cats" kaggle competition.

## Overview

This project uses transfer learning with a pretrained ResNet18 model to perform binary image classification. It's designed as an introduction to computer vision with PyTorch.

## Features

- Binary classification (cat or dog)
- Transfer learning using pretrained ResNet18
- Simple training pipeline
- Easy-to-use prediction script for new images

## Requirements

- Python 3.8+
- PyTorch
- torchvision
- Pillow
- matplotlib
